==============================================================
= Simple C Syntax Checker - Version 0.1                      =
==============================================================

Features:

	1. Multiple functions (more than one).
	2. Multiple statements (more than one).
	3. No functions and/or no statements.
	4. Simple function calls, no arguments.
	5. Include headers or none at all.
